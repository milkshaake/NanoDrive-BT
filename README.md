## Project Overview
A 4-wheel drive (4WD) robotic platform controlled via Bluetooth. This project served as an introduction to H-Bridge motor control and serial communication protocols.

## Hardware Architecture

Control: Arduino Nano

Communication: HC-05 Bluetooth (9600 baud)

Drive Train: 4x DC Motors via L298N Driver

Power: 7.4V Li-ion source

## Key Features

Real-time remote maneuvering via Android app.

PWM-based speed control for smooth acceleration.

Custom compact chassis featuring modular 3D-printed accessories, designed and optimized using SolidWorks and Autodesk Inventor

## Future Improvements

Integrating an ultrasonic sensor for autonomous obstacle avoidance.

Upgrading to a more efficient TB6612FNG motor driver to reduce heat and voltage loss.
